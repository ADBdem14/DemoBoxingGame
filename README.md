To run this repo (after cloning this):

Enter the Terminal with the following:

dotnet run dev

If you are looking for the exe of the game, follow these procedures. 

Folder of BoxingGame (click each folder/file):
bin > Release/net10.0win-x64 > publish

Arriving at the publish folder, you shall click the BoxingGame.exe to see and play the latest toggle work of the game.
